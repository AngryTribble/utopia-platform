# Utopia Platform

A modern Star Trek: Attack Wing digital companion platform.

## Initial Goal

Build the foundation for a modern card database and fleet builder for Star Trek: Attack Wing.

## MVP Scope

- Card database
- Card search and filters
- Basic fleet builder
- Point calculation
- Core legality checks
- Local fleet saving/export
- Easy card-data expansion

## Project Structure

apps/web        - Frontend web application
packages        - Shared engines and reusable code
